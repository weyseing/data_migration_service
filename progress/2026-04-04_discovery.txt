What it does:
  - Connects to source MySQL, inspects all tables/columns/types/FKs/indexes
  - Captures ENUM values (e.g. ENUM('pending','processing','shipped',...))
  - Extracts stored procedures via SHOW CREATE PROCEDURE
  - Builds FK dependency graph, outputs safe migration load order
  - Exposes results via REST API
  - Vue dashboard: stats, table overview, expandable details, SQL viewer, graph
